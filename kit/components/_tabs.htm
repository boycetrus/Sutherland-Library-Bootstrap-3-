Using Zinio on Windows 10 Tablets

Using Zinio for Libraries with Windows 8 / 10 devices involves a little more set up than with other platforms. In addition to your Zinio for Libraries account, you will require a Zinio.com account with the same email address as your Zinio for Libraries account.

Setting up the Zinio App for Offline Reading

  1. If you don't already have one, create a Zinio.com account using the same email address as your Zinio for Libraries account.
    Sign up at Zinio.com
  2. Download and install the Zinio App from the Microsoft App Store. Sign in to the app with your newly created credentials.
    Note: Windows 8 users may need to upgrade to Windows 10 to install the app.
     Get the App
  3. Add magazines to your collection from the Zinio for Libraries website using a web browser on your device to view and checkout individual issues.
    Choose Magazines
  4. Switch to the Zinio App after checking out magazines to download them to the app for offline reading.
Reading Online in the Browser

Windows 10 users can stream Zinio digital magazines in the browser from the Zinio for Libraries website without installing the Zinio App for Windows.
